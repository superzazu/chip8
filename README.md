# chip8

A Chip-8 interpreter written in C with the SDL2. Run `make` to build the project.

## Key bindings
| Computer key     | CHIP8 equivalent |
| ---------------- | ---------------- |
| 1                | 1                |
| 2                | 2                |
| 3                | 3                |
| 4                | C                |
| A                | 4                |
| Z                | 5                |
| E                | 6                |
| R                | D                |
| Q                | 7                |
| S                | 8                |
| D                | 9                |
| F                | E                |
| W                | A                |
| X                | 0                |
| C                | B                |
| V                | F                |
